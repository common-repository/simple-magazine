== Description ==

A simple Wordpress plugin to create magazine style issues.

Fork/contribute/give feedback at: 
[https://github.com/christofferok/simplemag](https://github.com/christofferok/simplemag)

== Installation ==

1. Search and install from the Plugins section in wp-admin

or 

1. Upload the `simplemag` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

= How do I create a new magazine? =

Click on Simple Magazine, then Issues. Here you create an issue which you will be able to add articles to.
